## ["Corrected Cat"](https://codehs.com/sandbox/karenlevinzon13/new-practice-pt-project-ti/run) Project !*!*
### About This Project !*!*
This program was created in JavaScript. It is a simple program, but it is also the first actual interactive game I have ever coded. I used lists, loops, conditionals, graphics, images, sequencing, procedures, parameters, timer events, and key-driven events to create this program. The purpose of this program was to create a simple but fun game that combines one of my favorite animals, a cat, and my hobby of playing games for fun, specifically guessing games, which I think are interesting! In this game, users have a few seconds to memorize a cat entirely composed of graphics. Then, the users are taken to another screen where the actual game begins. They get three options for the cat's ears, eyes, and so on, and must remember and select the correct one for each round until the cat is fully drawn. After the game is finished, the user receives a score out of 4, depending on how many of the full graphics they got correct across all 4 rounds, and if they drew the cat fully correct. Creating this game has taught me more about using loops, conditionals, and related constructs, and has given me the confidence to use them to create programs, such as simple games! I also learned that you can use these tools to solve problems, making the program's development much simpler and more efficient.
