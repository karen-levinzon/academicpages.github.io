# ["Guess the Song"](https://codehs.com/sandbox/karenlevinzon13/final-pt-project/run) Project !*!*
### About This Project !*!*
This is the second, and most recent, program I created using JavaScript. Similar to "Corrected Cat", this program also required the creation of lists, sequencing, procedures, parameters, algorithms, loops, conditionals, images, key-driven events, and timer events. However, the only difference is that this game includes many audio files. The purpose of this program was to create a fun way for players to test their knowledge of how many popular songs from each genre they know by guessing as many song titles correctly as possible. After the user begins the game, they are taken to a "Select a Game Mode" screen where they can choose from six genres to play. Then the user is asked to confirm their choice or go back to the selection screen to pick a new genre. Once the player selects their desired genre, the actual game begins. The player will have 10 seconds to listen to a randomly selected song before guessing its title. Whether they get the title right or wrong, a message confirming their result will be printed to the console before the user is asked whether they would like to continue playing with their current selection or go back to the selection screen to play with a new one. Afterward, the process repeats. Creating this program allowed me to learn more about using all the tools and skills needed to develop it, and how to apply them to make program development easier and more efficient. I also learned how to include audio files in my programs and how to use them. This helped improve what I already knew about JavaScript, allowing me to learn how to create new things that I might not have known I could.
